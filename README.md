This is a simple web-based Text-to-Speech (TTS) generator built using [Streamlit](https://streamlit.io/) and [Edge-TTS](https://github.com/rany2/edge-tts) (Microsoft TTS).

## Features
- Paste text into a textbox
- Select from different voice options (Male/Female)
- Adjust speech speed (Slow, Normal, Fast, Very Fast)
- Choose output format: `MP3`, `WAV`, `M4A`
- Download the generated audio file

## Requirements
- Python 3.8+
- pip

pip install -r requirements.txt
